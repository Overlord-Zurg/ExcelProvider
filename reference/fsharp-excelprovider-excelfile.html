ExcelProvider

ExcelFile

Namespace: FSharp.ExcelProvider

Typed representation of data in an Excel file.

Static parameters

Static parametersDescription
FileName
Signature: string

Location of the Excel file.
SheetName
Signature: string (optional, default = "")

Name of sheet containing data. Defaults to first sheet.
Range
Signature: string (optional, default = "")

Specification using A1:D3 type addresses of one or more ranges. Defaults to use whole sheet.
HasHeaders
Signature: bool (optional, default = true)

Whether the range contains the names of the columns as its first line.
ForceString
Signature: bool (optional, default = false)

Specifies forcing data to be processed as strings. Defaults to false.
